JZ XP LABEL v1.0

Enhanced, easy, functional and objective, this free all-Windows XP-style was made to compose varieties and effects at app design: BorderColor/HoverTime properties, MouseHover/MouseLeave and other. Zip kit contains a nicehand.cur and app with menus, onmouse, clicks, banners, as locked Text Box, etc.

Well, I wrote this control by using some few line codes from several authors at Internet whose credits I acknowledge e appreciate.

But the Module.Bas and the Subclass on separate mode to use MouseHover/MouseLeave events and HoverTime property was based on an Article from Source Guru, by HSPC (nickname). I credit to those persons all credits for these codes.

Properties

Alignment

Indicates disposition of Text in Box.

0 – Left

1 – Right

2 – Centralized

BackColor

Returns/Sets the Background color.

BorderColor

Returns/sets the Border color.

Caption

Returns/sets the text contained in the control.

Font

Returns a Font object.

HoverTime

Time to have elapsed before the MouseHover event action (user staying cursor positioned into control ambit.

HWnd

Returns a handle (from Microsoft Windows) to an object's window.

MouseIcon

Icon or Cursor picture will be the Mouse Pointer when MousePointer is setting to 99.

E.g. pick the "nicehand.cur" picture.

MousePointer

Returns/sets the type of mouse pointer displayed when over part of an object

E.g. set it to 99 mode recognize the MouseIcon you choose.

Other options will be in configured Windows collection.

TextColor

Returns/sets the Text color (on Caption).

Events

Change()

Occurs when the Caption contents are modified.

MouseDown(Button As Integer, Shift As Integer, X As Single, Y As Single)

Occurs when User picks down the mouse over control.

MouseHover()

Occurs when User pauses the Cursor over Control for defined time (see HoverTime property).

MouseLeave()

Occurs when Cursor gets out the control ambit.

MouseMove(Button As Integer, Shift As Integer, X As Single, Y As Single)

Occurs when the user moves the mouse to Control ambit.

Note: MouseMove starts immediately; MouseHover starts after delaying "HoverTime" milliseconds.

Enjoy!

How to use the JZ XP Label in your Application (for novices)
  1. Copy the source files JZXPLb.Ctl, MdlProc.bas and clsTrackInfo.cls to your project directory; and
  2. Open our project VB IDE
  3. Add MdlProc.bas as a existent Module;
  4. Add clsTrackInfo.cls as a existent Class; and
  5. Add JZXPLb.ctl as an existent UserControl (note the control icon will appear at Tool Window)
